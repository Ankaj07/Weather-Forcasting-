# 🌦️ INDRA Weather App

A responsive weather forecasting web application that provides **real-time weather updates** and **5-day weather forecasts** for any city using the **OpenWeather API**.

## 📖 Overview

INDRA Weather App is an academic web development project built to help users check current weather conditions and upcoming forecasts. The application features a clean and responsive interface, making it easy to access weather information from any device.

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💧 Humidity information
* 💨 Wind speed details
* ☁️ Weather condition display
* 📅 5-Day weather forecast
* 📱 Responsive user interface
* ⚡ Fast and simple navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeather API

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Ankaj07/Weather-Forcasting-.git
```

2. Navigate to the project folder:

```bash
cd Weather-Forcasting-
```

3. Add your OpenWeather API key in the JavaScript files if required.

4. Open `index.html` in your browser.
