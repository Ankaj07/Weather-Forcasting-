About our project:-



INDRA: A comprehensive weather forecasting platform developed as part of our academic coursework. This website provides real-time weather updates and forecasts for cities around the world specially India. Leveraging reliable APIs for data retrieval, it offers detailed weather information including temperature, humidity, wind speed, and a 5-day forecast. The user-friendly interface and responsive design ensure a seamless experience on both desktop and mobile devices. This project enhanced our backend and frontend development skills in web development, API integration, and user interface design.
